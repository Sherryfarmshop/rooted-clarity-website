# Rooted Clarity Health

Single-page scrolling website for [rootedclarityhealth.com](https://rootedclarityhealth.com).

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Before Launch

1. **Add your logo** — Place your tree-of-life logo at `public/images/logo.jpg`
2. **Connect email signup** — In `src/components/CTASignup.tsx`, wire the form to your email service (Mailchimp, ConvertKit, etc.)
3. **Update social links** — Verify URLs in `src/components/Footer.tsx`

## Tech Stack

- Next.js 16 (App Router)
- Tailwind CSS v4
- TypeScript
- Google Fonts: Playfair Display + Inter
